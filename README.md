# My Node.js Project

## Hướng dẫn Cài đặt

1. Clone dự án:
    ```bash
    git clone https://github.com/your-repo/my-node-project.git
    ```

2. Cài đặt các gói cần thiết:
    ```bash
    cd my-node-project
    npm install
    ```

3. Chạy ứng dụng:
    ```bash
    node src/app.js
    ```

## Các Chức Năng

- Lấy danh sách sách
- Tìm sách theo ISBN, tác giả, tiêu đề
- Lấy đánh giá sách
- Đăng ký và đăng nhập người dùng
- Thêm, chỉnh sửa và xóa đánh giá sách
